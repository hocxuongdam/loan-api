Introduction

Mini Aspire API

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with the Loan API.

Auth

Login

Example request:
curl --request POST \
    "http://localhost:8000/api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"duymai@gmail.com\",
    \"password\": \"password\"
}"

const url = new URL(
    "http://localhost:8000/api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "duymai@gmail.com",
    "password": "password"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "data": {
        "token": "bearer_token"
    },
    "success": {
        "message": "Success"
    }
}

Example response (401):


{
    "errors": "Unauthenticated"
}

List

requires authentication

List all current loans of the authenticated user

Example request:
curl --request GET \
    --get "http://localhost:8000/api/v1/loans?status=approved" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/v1/loans"
);

const params = {
    "status": "approved",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
